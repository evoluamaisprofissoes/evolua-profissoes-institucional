# Evolua+ Profissões — Site Institucional

Site institucional da **Evolua+ Profissões**, escola de cursos profissionalizantes localizada em Mirassol D'Oeste – MT.

## 🚀 Como publicar no GitHub Pages

1. Faça upload de todos os arquivos desta pasta para o repositório GitHub
2. Vá em **Settings → Pages**
3. Em **Source**, selecione `main` e pasta `/ (root)`
4. Clique em **Save**
5. Aguarde alguns minutos — o site estará disponível em `https://seu-usuario.github.io/nome-do-repositorio`

## 📁 Estrutura

```
/
├── index.html              → Página principal do site
├── uniforme_3307_x_4252_px_.png  → Logo/mascote Evolua+

```
